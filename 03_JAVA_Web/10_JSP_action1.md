## :bulb: Include, Forward 태그

| 이름    | 형식            | 설명                                                         |
| ------- | --------------- | ------------------------------------------------------------ |
| include | < jsp:include > | 이미 있는 jsp를 현재 jsp파일에 포함하는 태그                 |
| forward | < jsp:forward > | 서블릿에서 requestDispatcher 클래스의 포워딩 기능을 대신하는 태그 |
